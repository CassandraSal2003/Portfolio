Hangman Game

What is hangman?

Hangman is a guessing game where the user has to guess the correct word. The user can pick one letter at a time. If the user gets the letter wrong on a turn, a new part is added to the hang man. If they get the letter right, nothing is added to the hangman. The purpose of the game is to guess the word before completing the hangman fiugure. The program tells the user if they've won or if they've lost, and informs them of the correct word.

How?

I made this fairly simple but fun game with python.
