Banking Program

What is this?

"TTNY-Hackathon" is a collaborative project. This is my portion of the project, where the program prompts the user for information. The purpose of the project is to help students manage their savings and make a budget. The user can make budgets for different categories. They can deposit and withdraw money. When the user tries to withdraw, the system asks them which category they want to withdraw from. If what they withdraw is over their budget, the system warns the user.

How?

This project was made with python.
